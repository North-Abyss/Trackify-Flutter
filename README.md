# Trackify

A small Flutter habit-tracking demo app built while learning Flutter. The app shows a simple dashboard of habits, uses custom cards, and lets you toggle completion state by tapping each habit.

## Features

- Simple habit dashboard UI
- Custom `HabitCard` widget
- Tap a habit to toggle its completed state
- Material design styling with clean layout

## Getting Started

### Prerequisites

- Flutter SDK installed
- Dart SDK installed
- A device or emulator configured for Flutter

### Run the app

From the project root:

```bash
flutter pub get
flutter run
```

### Notes

- App entry point: `lib/main.dart`
- Main screen: `lib/screens/dashboard_screen.dart`
- Habit card widget: `lib/widgets/habit_card.dart`

## Project structure

- `lib/main.dart` - application entry point
- `lib/screens/dashboard_screen.dart` - dashboard and habit list
- `lib/widgets/habit_card.dart` - habit card UI component
